## 🖨️ 3D Printing (STL Files)
For those who don't have access to CNC machining, I've provided **STL files** for 3D printing.

### Recommended Print Settings
- **Material:** PETG or ABS (Recommended for durability), PLA+ is also fine.
- **Infill:** 40% or higher (Grid or Gyroid pattern for strength).
- **Wall Line Count:** 3-4 layers.
- **Support:** Required for some parts (Main Base and Rear Wheel Mount).
- **Layer Height:** 0.16mm - 0.2mm.

### Assembly Note
- Since 3D prints have slight shrinkage, you might need to clean up the holes with a **2mm drill bit** or a **reamer** for a perfect fit with the M2 bolts.
